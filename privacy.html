隐私政策

最后更新：2025年5月

律疏（以下简称「本应用」）尊重并保护用户隐私。本政策说明本应用如何处理您的数据。

一、我们不收集的数据

本应用不收集、不存储、不上传任何个人身份信息。我们没有服务器，不建立用户账号，不追踪您的使用行为。

二、本地存储的数据

以下数据仅存储于您的设备本地或您的个人iCloud账户中，我们无法访问：

· API Key：您配置的DeepSeek API Key加密存储于系统Keychain，可选同步至您的iCloud Keychain。

· 阅读记录：您的阅读位置和历史存储于iCloud Key-Value Store，仅您本人可访问。

· 对话记录：AI咨询的对话内容存储于iCloud Documents，仅您本人可访问。

· 应用设置：界面偏好等设置存储于本地UserDefaults。

三、AI咨询功能与第三方服务

使用AI法律顾问功能时，您的问题将直接从您的设备发送至DeepSeek（deepseek.com）的API接口。律疏不对该数据做任何中转、存储或分析。DeepSeek的数据处理方式请参阅其隐私政策。

四、法律数据

应用内置的法律法规数据来源于国家法律法规数据库，属于公开政府文件。这些数据完整存储于您的设备本地，无需联网即可访问。

五、儿童隐私

本应用不面向13岁以下儿童，不会故意收集儿童个人信息。
