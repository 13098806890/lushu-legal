常见问题

AI法律顾问功能为什么无法使用？

AI法律顾问功能需要配置DeepSeek API Key。请进入「设置」页面，在「DeepSeek API Key」处粘贴您的Key并点击「验证并保存」。DeepSeek新用户注册即有免费额度，价格低廉，国内可直连。

如何获取DeepSeek API Key？

访问 platform.deepseek.com，注册账号后在「API Keys」页面创建即可。新用户注册后有免费额度，无需付费即可体验。

我的API Key安全吗？

您的API Key仅加密存储于本机系统Keychain中（可选择同步至iCloud Keychain），不会上传至任何第三方服务器。AI咨询时您的问题直接发送至DeepSeek，律疏不做任何中转或存储。

法律数据需要联网吗？

不需要。1526部法律法规数据完整内置于App中，无需网络即可浏览全部法条和进行全文检索。仅AI咨询功能需要联网调用DeepSeek接口。

法律数据多久更新一次？

数据来源于国家法律法规数据库，收录截至App发布时的现行有效法律法规。更新将随App版本更新发布。

iCloud同步不工作怎么办？

请确认设备已登录iCloud账号，并在「系统设置 → Apple ID → iCloud」中确认「iCloud云盘」已开启。阅读进度和对话记录的同步依赖iCloud服务。

搜索时为什么找不到内容？

全文搜索需要至少输入3个字符才能触发条文内容搜索。建议使用具体的法律术语，例如「合同解除」「劳动报酬」等。也可在设置中关闭「仅搜索法律标题」以搜索条文内容。
